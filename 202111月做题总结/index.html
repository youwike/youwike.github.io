[USACO15JAN]Grass Cownoisseur G

link

code

标签：

缩点 + 拓扑排序

思路：

显然要先缩一下点，然后得到一个 dagdagdag ， 然后考虑在 dagdagdag 上做这道题。

根据 dagdagdag​ 的定义逆行后不会再经过逆行前的点，因为如果经过，那就说明从这个点能回到1点，但是从1点可以到达这个点，不符合 dagdagdag​ 的定义.

所以只需要对 111 所在的 sccsccscc 的前驱和后继分别跑一遍拓扑排序，然后通过逆行的边拼起来就行。即从 111 所在的 sccsccscc 开始跑后继，跑到一个点后逆行到前驱，然后从前驱回到 111, 显然只有走这样的路径才合法。

UVA11671 矩阵中的符号 Sign of Matrix

link

code

标签：

差分约数

思路：

考虑 xix_ixi​​​ 表示对于第 iii​​ 行的操作， yiy_iyi​​​ 表示对于第 iii​​ 列的操作。值为 +x+x+x​​ 表示加 xxx​, 值为 −x-x−x​ 表示减 xxx​​​。

对于第 iii 行, 第 jjj​​ 列的格子。如果为 +++ , 则 xi+yi>0x_i + y_i > 0xi​+yi​>0​ , 如果为 000 , 则 xi+yi=0x_i + y_i = 0xi​+yi​=0 , 如果为 −-− , 则 xi+yi<0x_i + y_i < 0xi​+yi​<0.

考虑把它们转化成差分约束的形式。

xi+yi>0⇔xi−(−yi)>0x_i + y_i > 0 \Leftrightarrow x_i - (-y_i) > 0xi​+yi​>0⇔xi​−(−yi​)>0​​​

xi+yi=0⇔xi−(−yi)=0x_i + y_i = 0 \Leftrightarrow x_i - (-y_i) = 0xi​+yi​=0⇔xi​−(−yi​)=0​​

xi+yi<0⇔−xi−yi>0⇔−yi−xi>0x_i + y_i < 0 \Leftrightarrow -x_i - y_i > 0 \Leftrightarrow -y_i - x_i > 0xi​+yi​<0⇔−xi​−yi​>0⇔−yi​−xi​>0​​​

然后跑差分约束（有负环就无解），但是差分约束出来的只是可行解，不保证最优，考虑让它们最优。

只需要所有变量同时减去一个 Δ\DeltaΔ​ , 原来的答案为 ∑i=1n∣xi∣+∑i=1n∣yi∣\sum_{i=1}^{n}|x_i| + \sum_{i=1}^{n}|y_i|∑i=1n​∣xi​∣+∑i=1n​∣yi​∣​ , 加上 Δ\DeltaΔ​ 后会变成 ∑i=1n∣Δ−xi∣+∑i=1n∣Δ−yi∣\sum_{i=1}^{n}|\Delta-x_i| + \sum_{i=1}^{n} |\Delta - y_i|∑i=1n​∣Δ−xi​∣+∑i=1n​∣Δ−yi​∣​

根据初中数学知识，当 Δ\DeltaΔ​ 为 所有 xix_ixi​​ 和 yiy_iyi​​​ 构成的数列的中位数时最优。（绝对值的几何意义）

然后就求出答案啦。
