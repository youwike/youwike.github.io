CF235D Graph Game

发表于 2021-10-02 更新于 2021-10-21

标签

期望 + 基环树

思路

先考虑树的做法, 对于两个点 uuu 和 vvv, vvv 在 uuu 的点分树内等价于 uuu 成为重心时 uuu 和 vvv 连通， 即 uuu 是 uuu 到 vvv 的路径上最早成为重心的点。显然概率为 1dis(u,v)+1\frac{1}{dis(u, v) + 1}dis(u,v)+11​, 那么答案就是 ∑u∈V∑v∈V1dis(u,v)+1\sum_{u\in V} \sum_{v \in V} \frac{1}{dis(u, v) + 1}∑u∈V​∑v∈V​dis(u,v)+11​

接着考虑拓展到基环树上，对于在同一棵树上的点，和树上的做法一样。对于不在同一棵树上的点，就要把环考虑进去，那么路径可以分为三个部分， 分别是两个半环和一条链，如图所示：

那么uuu 成为重心时 uuu 和 vvv 联通只需要 uuu 是 xxx 和 yyy 中最先成为重心的点或 uuu 是 xxx 和 zzz 中最先成为重心的点，这个概率是 1x+y+1+1x+z+1−1x+y+z\frac{1}{x + y + 1}+\frac{1}{x + z + 1}-\frac{1}{x + y + z}x+y+11​+x+z+11​−x+y+z1​, 即 uuu 是 xxx 和 yyy 中最先成为重心的点的概率加上 uuu 是 xxx 和 zzz 中最先成为重心的点的概率再减去 uuu 同时是 xxx , yyy, zzz 中最先成为重心的点的概率. 本质上是一个容斥。
