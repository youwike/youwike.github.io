[SHOI2008]仙人掌图 II题解

发表于 2021-10-21

[SHOI2008]仙人掌图II

标签

dp + 树形dp + 仙人掌 + 单调队列

思路

遇到仙人掌一般都会想到圆方树，然而这道题用不到。

考虑没有环的情况, 那就是树的直径了。

这道题是有环的，但是发现只有两个点都在环上时不能直接用求树的直径的方法转移。

设 fif_ifi​ ，对于环上的点表示 除非 iii 是当前环的根，否则 iii 不能经过环上的其它点，所能到达的最深的点。

对于不在环上的点，定义和树一样。

点的深度定义为到根节点的最短距离，环的根指这个环最先被 dfsdfsdfs 到的点。

那么当两个点都在环上时应该怎么转移呢？

根据 fif_ifi​ 的定义只有根可以从其它环上的点转移过来，从根的一个儿子开始绕环一圈并标号，设 cntcntcnt 表示环的大小, a[i]a[i]a[i] 表示环上标号为 iii 的点, 则 f[a[cnt]]=max⁡i=1cnt−1f[a[i]]+min⁡(cnt−i,i)f[a[cnt]] = \max_{i=1}^{cnt - 1}f[a[i]] + \min(cnt - i, i)f[a[cnt]]=maxi=1cnt−1​f[a[i]]+min(cnt−i,i)

然后就是如何统计跨环的链对于答案的影响。

先断环成链，然后枚举下标 iii,jjj 使 j<ij < ij<i 且 j≤i−cnt/2j \leq i - cnt / 2j≤i−cnt/2 因为 i,ji,ji,j 会选择较短的链.

ans′=max⁡(ans,f[a[i]]+f[a[j]]+i−j)ans' = \max(ans, f[a[i]] + f[a[j]] + i - j)ans′=max(ans,f[a[i]]+f[a[j]]+i−j) 这个东西可以用单调队列优化.

最后复杂度是 O(n)O(n)O(n) 的。
