    标签

    dpdpdp​ + lgvlgvlgv​ 引理 + 计数

    思路

    求从 (1,1)(1, 1)(1,1)​ 到 (n,n)(n, n)(n,n)​ 不相交，可以转化为 AAA​ 从 (1,2)(1,2)(1,2)​ 到 (n−1,n)(n - 1, n)(n−1,n)​ , BBB​ 从 (2,1)(2, 1)(2,1)​ 到 (n,n−1)(n, n - 1)(n,n−1)，路径不相交。 因为只有这样，才能使两人除起点和终点外路径不相交。

    用 lgvlgvlgv​​ 引理可解.

    Code

    总结

    两个人从 (1,1)(1, 1)(1,1) 到 (n,n)(n, n)(n,n) 可以考虑转化成 (1,2)(1,2)(1,2), (2,1)(2, 1)(2,1) 到 (n−1,n)(n - 1, n)(n−1,n) , (n,n−1)(n, n - 1)(n,n−1)​​​ ，这样做可以去除两个人起点和终点相同的影响

    B 小Z爱求和

    标签

    链表

    思路

    求每个区间第 kkk​​ 小的和， 可以转化为对于每个数，求它在多少个区间中是第 kkk​​ 小，把所有数从小到大排序，处理完一个数后把它删除，其它数的下标不变，这样可以保证序列中的所有数大于等于当前数。只需要从左边找 iii 个数，右边找 k−i−1k - i - 1k−i−1​ 个数， 那么在这些数中当前数一定是第 kkk​ 小。只要枚举 iii​​ 就可以了。删除操作可以用链表.

    Code

    总结

    求每个区间具有某个性质的数的和，可以对于每个数求一下它产生多少次贡献。

    和第 kkk 小有关的题，求每一个数的贡献，可以先把所有数从小到大排序，处理完一个数后把它删除。这样可以保证序列中的所有数大于等于当前数

    A 吊灯

    标签

    性质 + 卡常

    思路

    性质

    对于一棵树，它可以分成几个大小为 ddd 的联通块，当且仅当满足以下条件.

    1. d∣nd|nd∣n
    2. 有 nd\frac{n}{d}dn​ 个子树满足大小为 ddd​ 的倍数.

    充要性和必要性都显然. 因为每分出一个联通块，一定会减少一棵大小为 ddd​ 的子树，同时一些子树的大小变成 ddd ，如果找不到大小为 ddd​ 的子树，那么就不满足要求了.此时显然不能保证有 nd\frac{n}{d}dn​ 个子树满足大小为 ddd​​​ 的倍数.因为一定有至少一个点是不能分到联通块中的.

    枚举 ddd ,开一个桶 cnticnt_icnti​ 表示大小为 iii​ 的子树有多少个, 在 O(nd)O(\frac{n}{d})O(dn​) 的时间内可以求出大小为 ddd 的子树的个数.

    总复杂度为 (10×σ(n))(10 \times \sigma(n))(10×σ(n))​​，只有 707070​​ 分. 因为被卡常了.

    发现我们并不需要把树真正的建出来，这样可以省掉递归的常数，拿到 100100100 分。

    Code

    总结

    有时候，如果只需维护一些简单的信息，尤其是只从儿子向父亲转移，可以不用把树真正建出来。这样可以省掉递归的常数.
