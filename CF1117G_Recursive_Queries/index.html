发表于 2021-11-13 更新于 2021-11-14

标签

笛卡尔树 + 树状数组 + 线段树

思路

真是道笛卡尔树好题！

这个题显然问的是 [l,r][l, r][l,r] 构成的笛卡尔树每个节点的大小和。等于每个节点的深度和。事实上大小和，深度和都能做这道题，不过我写了深度和。

暴力建笛卡尔树是 O(nq)O(nq)O(nq)​ 的，过不去。考虑笛卡尔树的构建过程。可以用树状数组区间加，区间求和。来维护每个点的深度。

因为笛卡尔树的构建是插入式的，所以当插入到第 iii 个数时，笛卡尔树上恰好有 [1,i][1, i][1,i], 根据 f(l,r)=(r−l+1)+f(l,ml,r−1)+f(ml,r+1,r)f(l, r) = (r - l + 1) + f(l, m_{l, r} - 1) + f(m_{l, r} + 1, r)f(l,r)=(r−l+1)+f(l,ml,r​−1)+f(ml,r​+1,r) ，考虑把 f(l,r)f(l, r)f(l,r) 分成 f(l,ml,r+1)f(l, m_{l,r} + 1)f(l,ml,r​+1) 和 f(ml,r+1,r)f(m_{l, r} + 1, r)f(ml,r​+1,r) 来求，这样的好处是 ml,rm_{l, r}ml,r​ 有更多的性质，也就意味它在笛卡尔树上更容易找到。

至于寻找 ml,rm_{l, r}ml,r​​ 的方法，根据笛卡尔树的构建过程，构建笛卡尔树时栈里的点为从根一直走右儿子走到当前点所经过的所有的点。因为 ml,rm_{l, r}ml,r​ 为 [l,r][l, r][l,r] 内最大的点，所以 ml,rm_{l, r}ml,r​ 一定在栈内。在栈内二分找到第一个 ≥l\geq l≥l 的点即可，然后求区间和.

关于 ml,rm_{l,r}ml,r​ 在栈内的证明 :

假设 ml,rm_{l, r}ml,r​ 不在栈内，那它可以通过跳父亲一直跳到右链，它跳到的点无论下标还是权值都一定比它大，不符合 ml,rm_{l, r}ml,r​ 是 [l,r][l, r][l,r] 内最大的点.

求 f(l,ml,r+1)f(l, m_{l, r} + 1)f(l,ml,r​+1) 和 f(ml,r+1,r)f(m_{l, r} + 1, r)f(ml,r​+1,r) 的方法是对称的，会求其中一个另一个也就会了
